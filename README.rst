Install
------------------------------------------------------------------------------

``pynamodb_session_manager`` is released on PyPI, so all you need is to:

.. code-block:: console

    $ pip install pynamodb-session-manager

To upgrade to latest version:

.. code-block:: console

    $ pip install --upgrade pynamodb-session-manager
